# Plantar Pressure Monitoring System

## Overview

This project presents a real-time plantar pressure monitoring system designed for early detection of diabetic foot ulcer risk using Force Sensitive Resistors (FSRs) and Arduino R4 Minima.

The system continuously acquires pressure data across multiple plantar zones and performs real-time monitoring for abnormal pressure distribution patterns associated with ulcer development risk.

---

# Objectives

- Develop a low-cost embedded plantar pressure monitoring system
- Acquire real-time pressure data using FSR sensors
- Monitor pressure distribution across multiple foot regions
- Enable early identification of high-pressure ulceration zones
- Perform analytical processing of plantar pressure patterns

---

# Working Principle

The FSR sensors detect pressure variations across different plantar regions of the foot.

When force is applied:

- Sensor resistance changes
- Analog voltage varies
- Arduino reads analog sensor values
- Serial communication transmits real-time data
- Pressure values are analyzed for abnormal patterns
